# Matutino Genealogy & Ancestral Lineage Archive (Anilao, Iloilo)

This repository serves as the definitive, uncorruptible structural data network documenting the ancestral lineage and indigenous nobility (**Principalía por Sangre**) of the **Matutino family** of Anilao, Iloilo, Philippines.

## 📊 Core Datasets
* **`matutino_database.csv`**: The mathematically structured, graph-linked lineage database from Generation 1 (Felipe Matutino) to subsequent dynastic expansions, validating hereditary authority, endogamous alliances, and trans-epochal social continuity.

## 📜 Scientific Validation & Ground Truth
The algorithmic structure of this lineage is directly verified by, cross-referenced with, and grounded in the following peer-reviewed and open-access historical research:

* **Empirical Analysis Paper**: *Die Matutinos von Anilao: Eine Analyse indigener Nobilität auf Panay*  
  **Official DOI**: [https://doi.org/10.5281/zenodo.20278253](https://doi.org/10.5281/zenodo.20278253)
* **Primary Source Records**: Notarized land deeds, *Padron de Vecindario (1876)*, and *Relación de Reservados de Tributos (1853)*.
